Emotion Classification in Creative Text

[Code]

Investigated domain adaptation strategies for RoBERTa-base models on metaphorical and figurative texts (poetry and lyrics). Conducted an in-depth comparison and analysis of four finetuning strategies: classifier-head-only, partial unfreezing, full fine-tuning, and LoRA. 探究了基于 RoBERTa-base 模型在隐喻和修辞文本（诗歌和歌词）上的领域自适应策略。深入比较并分析了四种微调策略：仅微调分类头、部分解冻、全量微调和 LoRA。

Learning Dense Reward Functions in Chess

[Code]

Introduced an Inverse Reinforcement Learning (ML-IRL) framework to infer dense, intermediate reward signals from expert demonstrations (Grandmaster and Stockfish games). Developed a custom chess environment following the Gymnasium API with a residual convolutional reward model. 引入了一种逆向强化学习（ML-IRL）框架，从专家演示（特级大师和 Stockfish 对局）中推断密集的中间奖励信号。开发了一个遵循 Gymnasium API 的自定义国际象棋环境，并结合了一个残差卷积奖励模型。

Brain-to-Text: Decoding Intracortical Speech

[Code]

Developed a machine learning pipeline for the Brain-to-Text '25 competition to decode neural activity into natural language for patients with ALS. Implemented a two-stage hybrid Encoder-Decoder framework coupling a BERT-based encoder with a pre-trained GPT-2 decoder. Applied Masked Phoneme Modeling (MPM) and LoRA fine-tuning, achieving a Word Error Rate (WER) of 12.4%, and engineered a Day-Specific Adaptation Layer to handle non-stationary neural signals. 为 Brain-to-Text '25 竞赛开发了机器学习流水线，将 ALS 患者的神经活动解码为自然语言。实现了结合掩码音素建模（MPM）和 LoRA 微调的 BERT-GPT-2 混合框架，达到了 12.4% 的词错误率（WER），并设计了特定于日期的自适应层以处理非平稳神经信号。
